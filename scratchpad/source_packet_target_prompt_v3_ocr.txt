Selection order:
1. Identify which block's header/label best matches what the target field is asking for, then use the body of that block as the paste candidate.
2. If no header matches, pick the strongest relevant passage from `TEXT_BLOCKS`.
3. If a short single-value span in `EXACT_TEXT` is what the target needs (e.g. a title, price, name), use that instead.
4. Use `SCENE` only to understand what the source is, not as paste text.
